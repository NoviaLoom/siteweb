# Images Originales Appal Template

Ce dossier contient les images originales téléchargées depuis le site Marvel Theme (https://www.marveltheme.com/tf/html/appal/assets/images/).

## Structure des images téléchargées

### Banner
- `banner/bg-img-3.png` - Background principal du banner (22.7 KB)
- `banner/img-3.png` - Image ordinateur portable (86.3 KB)
- `banner/img-4.png` - Première image enfant (10.9 KB)
- `banner/img-5.png` - Deuxième image enfant (12.0 KB)

### Formes géométriques (Shapes)
- `shapes/cross.png` - Croix décorative
- `shapes/flow-1.png` - Forme de flux 1
- `shapes/circle-half.png` - Demi-cercle
- `shapes/box.png` - Boîte décorative
- `shapes/flow-2.png` - Forme de flux 2
- `shapes/circle.png` - Cercle complet

### Icônes (Icons)
- `icons/bg-0.png` à `icons/bg-8.png` - 9 backgrounds d'icônes colorés

### Images communes
- `map.png` - Image de carte du monde (20.5 KB)
- `map-mark.png` - Marqueur de carte
- `preloader.gif` - Animation de chargement (110 KB)
- `404.png` - Image page 404 (28.9 KB)

## Utilisation

Ces images originales peuvent être utilisées pour :
- Remplacer les images modifiées du template
- Créer des variations du design
- Développer de nouvelles sections
- Maintenir la cohérence avec le template original
